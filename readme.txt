**Arslania Acronym Kelimeler** eklentisi sayesinde yazilarinizda yer alan kisaltilmis kelimeleri,
acronym etiketiyle kolay bir sekilde kullanabilirsiniz.

== Description ==

Eklenti Ozellikleri:

* Kolay bir sekilde kelime ekleyebilir/silebilirsiniz.
* Kelimelerin gosterim sayisini da kaydeder.
* Kayitli kelimeleri duzenleyebilirsiniz.
* Yazi veritabanini **hicbir** sekilde etkilemez. 


== Installation ==

1. Eklentiyi indirdikten sonra zip dosyasindan cikarin.
2. Sitenizde yer alan **wp-content/plugins** dizini icerisine yukleyin.
3. WordPress yonetim panelinizde yer alan **Eklentiler** sekmesinden etkinlestirin.
4. **Ayarlar > Acronym Kelimeler** sekmesinden kelimelerinizi ekleyin/duzenleyin/silin.
